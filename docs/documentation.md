# SecureIncident

SecureIncident is a web-based incident reporting and management platform designed to help organizations securely record, track, and manage security-related incidents. The system provides a centralized solution for incident submission, user management, authentication, and incident monitoring through a modern web interface and REST API.

The platform consists of a React frontend, a FastAPI backend, and a PostgreSQL database, offering a secure, scalable, and user-friendly environment for handling incident reports.
