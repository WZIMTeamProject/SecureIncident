# Tests

SecureIncident includes automated tests for both the backend and frontend to ensure application reliability and maintainability.

## Backend Testing

Backend tests are written in Python using Pytest. They verify API endpoints, authentication workflows, database operations, business logic, and validation rules. Tests run in isolated database transactions to prevent persistent changes to development data.

## Frontend Testing

Frontend tests are written in TypeScript and focus on React components, user interactions, form validation, and application behavior. These tests help ensure that the user interface functions correctly and remains stable as new features are added.

Together, the backend and frontend test suites help maintain code quality and reduce the risk of regressions during development.
