# Backend

The SecureIncident backend is built with FastAPI, SQLAlchemy, and PostgreSQL. It provides REST API endpoints for authentication, user management, incident reporting, and password recovery.

## Features

* JWT-based authentication and authorization
* Incident creation and management
* User profile management
* Email-based password reset
* PostgreSQL database integration
* Alembic database migrations
* OpenAPI/Swagger API documentation

## Technologies

* FastAPI
* SQLAlchemy
* PostgreSQL
* Alembic
* Pydantic
* JWT Authentication
