# Frontend

The SecureIncident frontend is built with React, TypeScript, and Vite. It provides a responsive and user-friendly interface for authentication, incident reporting, and account management while communicating with the backend through REST APIs.

## Features

* User authentication and authorization
* Incident reporting and management interface
* User profile management
* Password reset workflow
* Responsive design
* API integration with the SecureIncident backend

## Technologies

* React 19
* TypeScript
* Vite
* Tailwind CSS 4
* React Router
* Axios/Fetch API
